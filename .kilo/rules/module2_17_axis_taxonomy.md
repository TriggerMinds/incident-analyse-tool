# MODULE 2: HET 17-ASSIGE TAXONOMIEMODEL & GEWEGEN BEWIJSMODEL

## 1. OVERZICHT VAN DE 17 ONAFHANKELIJKE ASSEN
1. **Ontstaanscontext:** Privé / Relationeel / Sociaal / Platformgebonden / Professioneel
2. **Deelnemersrol:** Handelen als zichzelf / Performer / Gecreëerde rol
3. **Productieorganisatie:** Informeel / Georganiseerd / Professioneel
4. **Productie-intentie:** Persoonlijk archief / Relationeel / Communicatie / Expressie / Commercie
5. **Publieksintentie:** Persoonlijk / Beperkt (1-op-1) / Communitygericht / Publiek
6. **Commerciële intentie:** Afwezig / Indirect / Gedeeltelijk / Primair
7. **Zichtbaarheidsoptimalisatie:** Schaal Niveau 1 (Geen externe verspreiding) t/m Niveau 7 (SEO/Conversie/Creators)
8A. **Industrie-onafhankelijk ontstaan (Sociale herkomst):** Volledig onafhankelijk ↔ Geïnitieerd door industrie/betaling
8B. **Industrie-onafhankelijke vormgeving (Stijl):** Niveau 1 (Geen industriële vorm) t/m Niveau 5 (Gedomineerd door porno-conventies)
9. **Camerabewustzijn:** Geen / Onzeker / Volledig bewust / Bewust zonder aanpassing / Expliciet / Publieksgericht
10. **Gedragsaanpassing:** Afwezig / Beperkt / Duidelijk / Volledig
11. **Spontaniteit:** Spontaan / Situationeel / Gedeeltelijk voorzien / Vooraf voorzien
12. **Planning:** Geen / Beperkt / Vooraf afgesproken / Voorbereid / Scenario
13. **Regie:** Ongeregisseerd / Informele afstemming / Sturing / Herhaling / Gescript
14. **Opname-afhankelijkheid (Counterfactual):**
    - Niveau 1: Opname-onafhankelijk (Handeling vond ook zonder camera plaats)
    - Niveau 2: Overwegend opname-onafhankelijk
    - Niveau 3: Gemengd
    - Niveau 4: Overwegend opname-afhankelijk
    - Niveau 5: Volledig opname-afhankelijk (Specifiek voor camera geënsceneerd)
    - Niveau 6: Niet vast te stellen
15. **Verspreidingsstatus:** Privé gebleven / Beperkt gedeeld / Vrijwillig publiek / Onvrijwillig gelekt (NCII) / Gemonetariseerd
16. **Authenticiteitsdimensie:** Contextueel / Esthetisch / Geclaimed / Geverifieerde oorsprong
17. **Vaststelbaarheid:** Geverifieerd / Waarschijnlijk / Onzeker / Niet vast te stellen

## 2. GEWEGEN BEWIJSMODEL & ANTI-DUBBELTELLINGSREGEL
- **Bewijssterkteschaal:** 0 (Geen), 1 (Zwak signaal - stijl), 2 (Matige aanwijzing), 3 (Sterke aanwijzing - 2 vereist voor exclusie), 4 (Direct/Onafhankelijk bevestigd bewijs - Paywall/Contract).
- **Anti-dubbeltellingsregel:** Gecorreleerde signalen uit één handeling (bijv. titel + 10 tags + thumbnail op 1 forum) worden GECONSOLIDEERD tot 1 enkele indicator op As 7 (Zichtbaarheidsoptimalisatie) met Score 2.

## 3. EINDVALIDATIESTEP (SELF-VALIDATION PROTOCOL - SVP)
Toets elke taalkundige of inhoudelijke conclusie verplicht aan de 4 SVP-vragen:
- **V1 (Letterlijke sprekerskans):** Zou een gemiddelde spreker dit letterlijk zo zeggen? [JA/NEE]
- **V2 (Oorsprongsdiagnose):** Afkomstig uit echte spontane communicatie of formele beschrijving? [JA/NEE]
- **V3 (AI-Normalisatie check):** Is het een AI-opschoning/sterilisatie? [JA/NEE]
- **V4 (Variant-dominantie):** Is er een natuurlijkere variant die vaker voorkomt? [JA/NEE]
