## Methodologisch Synthesedocument: Het Meerassige Classificatiekader voor Intieme Media

Dit document operationaliseert het uitgebreide methodologische classificatiekader. Het ontkoppelt binaire aannames en vervangt enkelvoudige hoofdlabels door een **15-assige multidimensionale taxonomie**. 

Het uitgangspunt hierbij is **epistemologische hygiëne**: kenmerken zoals spontaniteit, authenticiteit, toestemming en commerciële intentie mogen nooit worden afgeleid uit puur stilistische kenmerken (zoals beeldkwaliteit, schokkerige cameravoering of locatie), maar moeten expliciet worden geanalyseerd per afzonderlijke variabele.

---

### 1. Ontkoppeling van de Kernvariabelen

Het kader ontkoppelt variabelen die in traditioneel onderzoek vaak ten onrechte zijn samengevoegd:

```
+------------------------+      +------------------------+      +------------------------+
|    CAMERABEWUSTZIJN    |      |      SPONTANITEIT      |      |        PLANNING        |
| Mate waarin men weet dat|  ≠   | Mate waarin opname in  |  ≠   | Mate van praktische    |
| opname plaatsvindt.    |      | moment zelf ontstaat.  |      | voorbereiding vooraf.  |
+------------------------+      +------------------------+      +------------------------+
            ≠                               ≠                               ≠
+------------------------+      +------------------------+      +------------------------+
|         REGIE          |      |    PUBLIEKSINTENTIE    |      |  COMMERCIËLE INTENTIE  |
| Sturing, herhaling en  |  ≠   | Beoogde doelgroep voor |  ≠   | Aanwezigheid van een   |
| scripting van gedrag.  |      | de opname.             |      | verdienmodel/gewin.    |
+------------------------+      +------------------------+      +------------------------+
```

* **Camerabewustzijn vs. Enscenering:** Iemand kan zich 100% bewust zijn van de camera en direct in de lens kijken (bijv. een intieme selfie) zonder dat er sprake is van een geregisseerde, gescripte of commerciële enscenering.
* **Publieksintentie vs. Commerciële intentie:** Het bewust delen van intieme media op een openbaar forum of binnen een gemeenschap (publieke zelfexpressie) is analytisch iets anders dan het verkopen of monetariseren van die beelden (commerciële intentie).
* **Productie-intentie vs. Verspreidingsstatus:** De oorspronkelijke reden van herkomst (bijv. relationele communicatie) blijft ongewijzigd, ongeacht of het bestand later privé blijft, vrijwillig geüpload wordt of via een datalek onvrijwillig in het publieke domein belandt.

---

### 2. Het Volledige 15-Assige Taxonomiemodel

Elk mediabestand wordt niet beoordeeld op basis van één overkoepelende term, maar krijgt een **profiel van 15 afzonderlijke variabelen**:

| As | Variabele | Mogelijke categorieën / Schaalwaarden |
| :--- | :--- | :--- |
| **1** | **Productiecontext** | Privé / Relationeel / Sociaal / Platformgebonden / Professioneel |
| **2** | **Professionalisering** | Informeel / Semi-professioneel / Professioneel |
| **3** | **Commerciële intentie** | Afwezig / Indirect (reputatie) / Gedeeltelijk / Primair (verdienmodel) |
| **4** | **Publieksintentie** | Persoonlijk / Beperkt (1-op-1 of besloten groep) / Communitygericht / Publiek |
| **5** | **Spontaniteit** | Onverwacht / Situationeel / Gedeeltelijk voorzien / Volledig vooraf voorzien |
| **6** | **Planning** | Geen voorbereiding / Beperkt praktisch / Vooraf afgesproken / Voorbereid / Scenario |
| **7** | **Regie** | Ongeregisseerd / Informele afstemming / Gedeeltelijke sturing / Herhaling / Expliciete regie / Gescript |
| **8** | **Camerabewustzijn** | Geen / Onzeker / Volledig bewust / Bewust zonder aanpassing / Gedeeltelijk reagerend / Expliciet gedrag / Publieksgericht |
| **9** | **Deelnemersconfiguratie** | Individueel / Partners / Groep / Derde als filmer |
| **10** | **Opnamemethode** | Zelfopname (selfie/POV) / Opname door deelnemer / Bekende derde / Externe producent |
| **11** | **Oorspronkelijk doel** | Herinnering / Communicatie / Experiment / Expressie / Performance / Commercie |
| **12** | **Verspreidingsstatus** | Privé gebleven / Beperkt gedeeld / Vrijwillig publiek / Gemonetariseerd / Onvrijwillig verspreid (NCII) |
| **13** | **Industriële invloed** | Afwezig / Indirect (nabootsing) / Stilistisch / Structureel (seksindustrie) |
| **14** | **Authenticiteitsclaim** | Geen claim / Persoonlijk / Spontaan / Amateur / Echt / Documentaire claim |
| **15** | **Vaststelbaarheid** | Geverifieerd / Waarschijnlijk / Onzeker / Niet vast te stellen |

### 3. Logica van Terminologische Toewijzing (Post-Classificatie)

Pas **nadat** alle 15 assen zijn gescoord, wordt op basis van de specifieke waarden een beschrijvende term toegewezen. 

```
                                 [ MEE RASSIGE EVALUATIE ]
                                 (Scoren van Assen 1 t/m 15)
                                             |
     +---------------------------------------+---------------------------------------+
     |                                       |                                       |
[ Privé + Geen Commercie + Zelfopname ]  [ Publiek + Geen Commercie + Community ]  [ Commercie + Amateur-esthetiek ]
     |                                       |                                       |
     v                                       v                                       v
"Privately produced self-recorded       "Publicly shared non-commercial          "Commercial amateur-style
       intimate media"                          intimate media"                         content"
```

#### Regels voor terminologische toewijzing:
* **"privately produced intimate media"**: Vereist dat As 1 = *Privé/Relationeel*, As 2 = *Informeel*, en As 12 = *Privé gebleven of beperkt gedeeld*.
* **"non-commercial intimate media"**: Vereist dat As 3 = *Afwezig*.
* **"self-recorded intimate media"**: Vereist dat As 10 = *Zelfopname*.
* **"consensually created intimate media"**: Vereist dat As 15 betrekking op toestemming *Geverifieerd* is via bronnen (en niet louter verondersteld uit de beelden).
* **"publicly shared non-commercial intimate media"**: Vereist dat As 4 = *Publiek*, As 3 = *Afwezig* en As 12 = *Vrijwillig publiek*.
* **"commercial amateur-style content"**: Vereist dat As 3 = *Primair* en As 13 = *Stilistisch of Structureel*, ondanks een eventuele informele uitstraling.

* **Exclusieregel voor "Amateur sexual media":** Deze term wordt **niet** gebruikt als primaire analytische classificatie, omdat deze voorbijgaat aan de cruciale verschillen op As 3 (Commercie), As 7 (Regie) en As 13 (Industriële invloed).

### 4. Demonstratie: Verplichte Rapportering per Casus

Hieronder worden vier casussen geanalyseerd conform het verplichte 7-punts rapporteringsformaat.

---

#### Casus A: Spontane relationele intieme opname tijdens vakantie

```
[Visuele kenmerken: Smartphone-video op een hotelkamer, lage belichting, lachende personen]
```

##### 1. Waargenomen kenmerken
* Bewegend beeld opgenomen in een hotelkamer. Twee personen interacteren intiem. Eén persoon houdt de smartphone in de hand.
* Lachen en informele spraak hoorbaar. Geen herhaling van handelingen waarneembaar.

##### 2. Classificatie per afzonderlijke as
1. *Productiecontext:* Relationeel
2. *Professionalisering:* Informeel
3. *Commerciële intentie:* Afwezig
4. *Publieksintentie:* Beperkt (1-op-1)
5. *Spontaniteit:* Situationeel ontstaan
6. *Planning:* Geen voorbereiding
7. *Regie:* Ongeregisseerd
8. *Camerabewustzijn:* Bewust van de opname, maar zonder zichtbare gedragsaanpassing
9. *Deelnemersconfiguratie:* Partners
10. *Opnamemethode:* Opname door deelnemer
11. *Oorspronkelijk doel:* Relationele herinnering
12. *Verspreidingsstatus:* Privé gebleven (opgeslagen op lokale hardware)
13. *Industriële invloed:* Afwezig
14. *Authenticiteitsclaim:* Persoonlijk
15. *Vaststelbaarheid:* Geverifieerd (via interview met makers)

##### 3. Toegewezen term & Argumentatie
* **Term:** *Privately produced non-commercial intimate media*
* **Argumentatie:** De productie vond plaats binnen een besloten relationele context zonder commerciële intentie, planning of regie, en is privé gebleven.

##### 4. Direct waarneembare eigenschappen
* Twee personen op hotelkamer, gebruik van consumentensmartphone, informele spraak, fysieke interactie.

##### 5. Uit bronnen blijkende eigenschappen
* Verificatie dat de opname door de partners zelf is gehouden en nooit aan derden is verzonden.

##### 6. Slechts veronderstelde eigenschappen
* Geen (volledig geverifieerd via bron).

##### 7. Niet betrouwbaar vast te stellen classificaties
* Geen; alle variabelen zijn onderbouwd.

---

#### Casus B: Intieme spiegel-selfie verzonden via messaging-app

```
[Visuele kenmerken: Hoge resolutie foto, geposeerd voor spiegel, doordachte belichting]
```

##### 1. Waargenomen kenmerken
* Stilstaand naaktbeeld van één persoon, genomen via een badkamerspiegel.
* Posering is esthetisch ingesteld (belichting van opzij).

##### 2. Classificatie per afzonderlijke as
1. *Productiecontext:* Privé / Relationeel
2. *Professionalisering:* Informeel
3. *Commerciële intentie:* Afwezig
4. *Publieksintentie:* Beperkt (één specifieke ontvanger)
5. *Spontaniteit:* Gedeeltelijk voorzien
6. *Planning:* Voorbereide handelingen of posities (aandacht voor hoek)
7. *Regie:* Informele onderlinge afstemming (zelf-sturing)
8. *Camerabewustzijn:* Expliciet gedrag voor de camera
9. *Deelnemersconfiguratie:* Individueel
10. *Opnamemethode:* Zelfopname (selfie)
11. *Oorspronkelijk doel:* Unterling communicatie / Intieme interactie
12. *Verspreidingsstatus:* Eén-op-één gedeeld
13. *Industriële invloed:* Indirect (visuele stijl beïnvloed door online esthetiek)
14. *Authenticiteitsclaim:* Persoonlijk
15. *Vaststelbaarheid:* Waarschijnlijk (gebaseerd op metadata en platformsyntax)

##### 3. Toegewezen term & Argumentatie
* **Term:** *Self-recorded non-commercial intimate media*
* **Argumentatie:** De opname is door de betrokkene zelf gemaakt (selfie) voor een specifieke ontvanger. De aanwezigheid van posering (As 6/7) maakt het niet commercieel, aangezien As 3 afwezig is.

##### 4. Direct waarneembare eigenschappen
* Spiegel-selfie, één persoon, geposeerd, badkameromgeving.

##### 5. Uit bronnen blijkende eigenschappen
* EXIF-metadata bevestigt smartphone-hardware en tijdstempel.

##### 6. Slechts veronderstelde eigenschappen
* De exacte relationele dynamiek met de ontvanger op het moment van verzenden.

##### 7. Niet betrouwbaar vast te stellen classificaties
* Toestemming van de ontvanger om de foto te bewaren kan niet uit de foto zelf worden afgeleid.

#### Casus C: OnlyFans-creatorvideo in "thuisstijl"

```
[Visuele kenmerken: Video opgenomen in een huiskamer op een statief, 'amateur/unfiltered' stijl, directe aanspraak van de kijker]
```

##### 1. Waargenomen kenmerken
* Bewegend beeld van een persoon in een woonkamer. Directe oogcontact met de camera, spreken tegen de kijker ("Hey guys").
* Geen professionele belichtingsset, wel een statief opname.

##### 2. Classificatie per afzonderlijke as
1. *Productiecontext:* Platformgebonden / Semi-professioneel
2. *Professionalisering:* Semi-professioneel
3. *Commerciële intentie:* Primair (geplaatst achter betaalmuur)
4. *Publieksintentie:* Communitygericht / Publiek (betalende abonnees)
5. *Spontaniteit:* Volledig vooraf voorzien
6. *Planning:* Voorbereide handelingen en setting
7. *Regie:* Gedeeltelijke sturing / Herhaling
8. *Camerabewustzijn:* Volledig camera- en publieksgericht gedrag
9. *Deelnemersconfiguratie:* Individueel
10. *Opnamemethode:* Zelfopname / Statief
11. *Oorspronkelijk doel:* Commerciële exploitatie
12. *Verspreidingsstatus:* Gemonetariseerd (Paywall)
13. *Industriële invloed:* Stilistisch en structureel
14. *Authenticiteitsclaim:* "Amateur" / "Echt" (Marketingclaim)
15. *Vaststelbaarheid:* Geverifieerd (Platform-URL en verdienmodel vastgesteld)

##### 3. Toegewezen term & Argumentatie
* **Term:** *Commercial amateur-style content*
* **Argumentatie:** Hoewel de visuele stijl "thuisgemaakt" oogt (As 2/14), sturen het primaire verdienmodel (As 3 = Primair) en het publieksgerichte gedrag (As 8) de classificatie naar commerciële content in amateurstijl.

##### 4. Direct waarneembare eigenschappen
* Woonkamer, statiefopname, directe aanspraak van de kijker, intieme handelingen.

##### 5. Uit bronnen blijkende eigenschappen
* Pagina-URL op een abonnementsplatform (OnlyFans), prijs per maand vastgesteld via platform.

##### 6. Slechts veronderstelde eigenschappen
* De mate waarin de gepresenteerde emoties overeenkomen met de persoonlijke beleving van de maker.

##### 7. Niet betrouwbaar vast te stellen classificaties
* Het exacte netto-inkomen verkregen uit dit specifieke bestand.

#### Casus D: Privé geproduceerde video, later ongeautoriseerd gelekt (NCII)

```
[Visuele kenmerken: Korrelige video, slaapkamer, twee bekende personen, gepubliceerd op een openbaar video-forum met de titel "LEAKED PRIVATE TAPE"]
```

##### 1. Waargenomen kenmerken
* Privé-opname van twee personen. Geen logo's of commerciële intro's.
* Bestand aangetroffen op een openbare verzamelsite.

##### 2. Classificatie per afzonderlijke as
1. *Productiecontext:* Relationeel (Oorspronkelijk)
2. *Professionalisering:* Informeel
3. *Commerciële intentie:* Afwezig (bij productie) / Onbekend (bij verspreider)
4. *Publieksintentie:* Uitsluitend voor de aanwezige betrokkenen (Oorspronkelijk)
5. *Spontaniteit:* Situationeel ontstaan
6. *Planning:* Vooraf afgesproken opname
7. *Regie:* Ongeregisseerd
8. *Camerabewustzijn:* Volledig bewust van de opname
9. *Deelnemersconfiguratie:* Partners
10. *Opnamemethode:* Opname door deelnemer
11. *Oorspronkelijk doel:* Relationele herinnering
12. *Verspreidingsstatus:* Onvrijwillig verspreid (NCII / Buiten oorspronkelijke doelgroep)
13. *Industriële invloed:* Afwezig (bij productie)
14. *Authenticiteitsclaim:* Geen claim van maker / Wel exploitatie-claim door uploader
15. *Vaststelbaarheid:* Waarschijnlijk (verklaring van slachtoffer/politierapport)

##### 3. Toegewezen term & Argumentatie
* **Term:** *Non-consensually disseminated private intimate media*
* **Argumentatie:** De oorspronkelijke productie-intentie (As 11 = Relationele herinnering; As 3 = Geen commercie) blijft bepalend voor het herkomstprofiel. De verspreidingsstatus (As 12 = Onvrijwillig verspreid) bepaalt het misbruikkarakter.

##### 4. Direct waarneembare eigenschappen
* Slaapkameromgeving, twee personen, informeel cameragebruik.

##### 5. Uit bronnen blijkende eigenschappen
* Melding bij helpdesk/politieaangifte waaruit blijkt dat de video zonder toestemming van de cloud is gehaald.

##### 6. Slechts veronderstelde eigenschappen
* De identiteit en het motief van de anonieme uploader.

##### 7. Niet betrouwbaar vast te stellen classificaties
* Of de uploader financieel gewin heeft gehad bij het herplatseen op de externe site (As 3 voor verspreider).

### 5. Epistemologisch Niet-herleidbaarheidsbeginsel

Het classificatiekader hanteert een strikt **niet-herleidbaarheidsbeginsel**:

> **Regel:** Bepaalde ethische, juridische en sociologische variabelen kunnen **nooit** uitsluitend worden afgeleid uit de visuele of audiomerken van het mediabestand zelf.

| Kenmerk | NIET af te leiden uit... | WEL vast te stellen via... |
| :--- | :--- | :--- |
| **Authenticiteit** | Lage beeldkwaliteit, schokkerige camera, rommelige slaapkamer. | Verificatie van de productiecontext en maker-historie. |
| **Spontaniteit** | Lachen, onvoorspelbare bewegingen, natuurlijke spraak. | Onafhankelijke analyse van de voorbereidingsfase. |
| **Toestemming (Consent)** | Glimlachen, actieve deelname, visuele interactie op beeld. | Expliciete, externe verificatie van de overeenkomst tussen betrokkenen. |
| **Niet-commercieel karakter** | Het ontbreken van logo's, Watermerken of HD-montage. | Analyse van de distributiekanalen en financiële stromen. |

Door dit principe toe te passen, blijft het onderzoek gefundeerd op waarneembare feiten en expliciete bronnen, en wordt voorkomen dat commerciële vervalsingen of ethische inbreuken onjuist worden geanalyseerd.
