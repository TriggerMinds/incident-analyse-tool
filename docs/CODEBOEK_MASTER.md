# CODEBOEK MASTER — Referentiedocument

Dit bestand is een verwijzing naar de operationele codeboek-definitie.

**Actieve definitie:** `.kilo/rules/master_codebook.md`

Dit bestand bevat de integrale definities van het Geïntegreerd Gedragsmatig Codeboek (Domein A t/m P) voor digitaal-sociologisch onderzoek naar jongeren (15–18 jaar). Raadpleeg het actieve bestand voor de meest actuele velden, waarden en syntax.

## Samenvatting van de 16 domeinen

| Domein | Record | Beschrijving |
|---|---|---|
| A | `PopulationProfile` | Leeftijd, geslacht, positie, regio |
| B | `ContextVector` | Locatie (event/network/harm), tijd, sociale relatie |
| C | `BehaviouralContentRecord` | Zichtbaarheid, gedragsklasse, opnamewijze |
| D | `VisualScriptRecord` | Sociaal script, motief, fase |
| E | `NormalizationContextRecord` | Groepsnormen, prevalentie, grenzen |
| F | `DeviceAffordanceRecord` | Apparaat, platform, technische mogelijkheden |
| G | `MediaLifecycleRecord` | Capture, storage, first-share, re-share |
| H | `TrajectoryRecord` | Volledige 9-fasen keten |
| I | `VisibilityIntentMismatchRecord` | Mismatches tussen bedoeling en realiteit |
| J | `ConsentByActionRecord` | Toestemming per handeling (5 niveaus) |
| K | `ControlLossRecord` | Verlies van controle, omkeerbaarheid |
| L | `ResponseAndRecoveryRecord` | Reactie, impact, herstel |
| M | `DeltaFieldRecord` | 10 Delta-inzichten geoperationaliseerd |
| N | `MissingnessMap` | Datakwaliteit per variabele |
| O | `EvidenceProfile` | Bron, granulariteit, noemer |
| P | `NonEventBaseline` | Escalatie-vrije referentiecasus |

Zie `.kilo/rules/master_codebook.md` voor volledige velddefinities.
